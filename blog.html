Show the difference between localStorage and sessionStorage

localStorage

1.The storage capacity of local storage is 5MB/10MB

2.As it is not session-based, it must be deleted via javascript or manually

3.The client can only read local storage

4.There is no transfer of data to the server

SessionStorage

1.The storage capacity of session storage is 5MB

2.It's session-based and works per window or tab. This means that data is stored only for the duration of a session, i.e., until the browser (or tab) is closed

3.The client can only read local storage

4.There is no transfer of data to the server

How does the javascript event loop work?

The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack. Once the Stack is clear, the event loop triggers and checks the Task Queue for available callbacks. If there are any, it pushes it to the Call Stack, waits for the Call Stack to be clear again, and repeats the same process.
